# TrendScope PRD

## AI 기반 뉴스 키워드 트렌드 분석 플랫폼

---

# 1. 프로젝트 개요

## 프로젝트명

TrendScope

## 한줄 소개

사용자가 관심 키워드를 등록하면 네이버 뉴스 데이터를 수집·분석하여 현재 가장 뜨거운 이슈와 관련 키워드를 시각화해주는 웹 플랫폼

---

# 2. 프로젝트 목표

본 프로젝트의 목표는 다음과 같다.

* 사용자가 관심 있는 키워드의 뉴스 흐름을 쉽게 파악할 수 있도록 한다.
* 뉴스 헤드라인 데이터를 기반으로 현재 가장 많이 언급되는 이슈를 분석한다.
* 데이터 시각화를 통해 트렌드 변화를 직관적으로 제공한다.
* REST API 기반 풀스택 구조를 구현한다.
* 외부 API(Naver News API)를 연동한다.

---

# 3. 핵심 기능

## 3.1 회원 기능

### 회원가입

* 이메일 기반 회원가입
* 비밀번호 암호화 저장

### 로그인

* JWT 기반 인증
* 로그인 상태 유지

### 마이페이지

* 등록 키워드 조회
* 관심 키워드 관리

---

# 3.2 키워드 관리

사용자는 관심 키워드를 등록할 수 있다.

예시:

* AI
* 삼성전자
* 이준석
* 반도체

기능:

* 키워드 추가
* 키워드 삭제
* 키워드 수정
* 키워드 활성화/비활성화

---

# 3.3 뉴스 수집

## 외부 API

Naver News Search API 사용

수집 방식:

* 사용자 키워드 기반 검색
* 헤드라인 중심 데이터 수집
* 최대 300개 기사 수집

수집 데이터:

* 제목
* 링크
* 언론사
* 게시 시간
* 요약문(optional)

---

# 3.4 트렌드 분석

수집된 뉴스 데이터를 기반으로 분석 수행

분석 항목:

* 가장 많이 등장한 단어
* 급상승 키워드
* 연관 키워드
* 시간대별 언급량
* 기사 수 변화 추이

---

# 3.5 데이터 시각화

## 대시보드 제공

### 워드 클라우드

많이 등장한 단어를 시각적으로 표현

### 트렌드 그래프

시간대별 기사 수 변화

### 관련 키워드 그래프

연관 단어 시각화

### 오늘의 감자

현재 가장 많이 언급되는 핵심 이슈 표시

---

# 3.6 AI 요약 기능 (선택 기능)

OpenAI API 또는 로컬 LLM 기반

기능:

* 뉴스 흐름 요약
* 핵심 이슈 요약
* 관련 사건 설명

예시:

> "오늘 삼성전자 관련 뉴스는 AI 반도체 투자 확대와 실적 발표 이슈가 중심입니다."

# 7. 데이터베이스 설계

## User

| 컬럼       | 설명       |
| -------- | -------- |
| id       | PK       |
| email    | 이메일      |
| password | 암호화 비밀번호 |
| nickname | 닉네임      |

---

## Keyword

| 컬럼         | 설명     |
| ---------- | ------ |
| id         | PK     |
| user_id    | 사용자 FK |
| keyword    | 키워드명   |
| created_at | 생성일    |

---

## News

| 컬럼           | 설명     |
| ------------ | ------ |
| id           | PK     |
| keyword_id   | 키워드 FK |
| title        | 뉴스 제목  |
| link         | 기사 링크  |
| publisher    | 언론사    |
| published_at | 게시 시간  |

---

## TrendAnalysis

| 컬럼          | 설명     |
| ----------- | ------ |
| id          | PK     |
| keyword_id  | 키워드 FK |
| top_keyword | 핵심 키워드 |
| issue_score | 이슈 점수  |
| analyzed_at | 분석 시간  |

---

# 8. API 설계 예시

## 인증 API

### 회원가입

```http
POST /api/auth/signup
```

### 로그인

```http
POST /api/auth/login
```

---

## 키워드 API

### 키워드 등록

```http
POST /api/keywords
```

### 키워드 목록 조회

```http
GET /api/keywords
```

### 키워드 삭제

```http
DELETE /api/keywords/{id}
```

---

## 뉴스 API

### 뉴스 분석 요청

```http
POST /api/news/analyze
```

### 뉴스 결과 조회

```http
GET /api/news/{keywordId}
```
